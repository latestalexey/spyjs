Порядок проведения эксперимента:
  1. поместите в первое поле исходный текст;
  2. нажмите кнопку "Начать";
  3. во втором текстовом поле наберите указанный выше текст;
  4. нажмите кнопку "Закончить";
  5. повторите этапы 2-4 для каждого испытуемого;
  6. нажмите кнопку "Идентификация" для оценивания схожести стилей набора текста.
Примечание: в рамках одного эксперимента исходный текст должен оставаться неизменным.
